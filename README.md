# Flutter Hackathon
# イベント探しアプリ
## 概要
コロナ収束後に開催されるイベントを盛り上げるアプリ

## 使用技術
- Flutter
- Riverpod
- Google Map
- Freezed
- Firebase (auth, storage, firestore)
- github actions
